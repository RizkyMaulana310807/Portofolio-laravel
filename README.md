# 🚀 Auto Scroll Carousel

Auto Scroll Carousel adalah komponen JavaScript sederhana untuk membuat scroll horizontal otomatis (infinite) dengan fitur pause saat hover.

---

## 📸 Preview

![Preview](https://via.placeholder.com/800x200?text=Auto+Scroll+Preview)

---

## ✨ Fitur

* 🔁 Infinite horizontal scroll
* ⏸ Pause saat hover
* ⚡ Ringan tanpa library
* 📱 Support responsive

---

## 📦 Instalasi

Cukup clone repo ini:

```bash
git clone https://github.com/username/auto-scroll-carousel.git
cd auto-scroll-carousel
```

---

## 🛠 Cara Pakai

### HTML

```html
<div id="track" style="overflow-x: auto; white-space: nowrap;">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
</div>
```

### JavaScript

```javascript
import { scroll } from "./scroll.js";

scroll();
```
